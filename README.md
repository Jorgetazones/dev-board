# 🚀 Dev-Board: Productivity Dashboard
Proyecto creado con **React + TypeScript + Vite** para la gestión eficiente de tareas técnicas.

## ✨ Características
- 📝 Gestión de tareas con prioridades.
- 📊 Estadísticas en tiempo real (pendientes/completadas).
- 💾 Persistencia de datos en LocalStorage.
- 🛠️ Tipado estricto con TypeScript.

## 🛠️ Instalación
```bash
yarn install
yarn dev
---
### 🚀 Tu siguiente movimiento

Una vez tengas el repo conectado, cada vez que termines una función (como el `addTask` o el `deleteTask`), haz esto:
1. `git add .`
2. `git commit -m "feat: added task deletion logic"`
3. `git push`

**¿Ya has creado el repositorio en GitHub?** Si es así, pásame el código de tu `App.tsx` o de tu interface de `Task` y empezamos a picar la lógica para que ese primer commit tenga contenido de calidad. 💻🔥
